# Motivation and Scope

Bring Flags binary operation to Class Enum, and allow developers using Class Enum as Flags.

As developer of C#, Java, Rust or  i am quite accustomed to working with Flags variable using Tenarry operators, for example here is C#

```C#
using System;

[Flags]
public enum ProductPermissions
{
    None    = 0,         // Binary: 0000 (Always include a 0 value)
    Read    = 1 << 0,    // Binary: 0001 (Decimal 1)
    Write   = 1 << 1,    // Binary: 0010 (Decimal 2)
    Execute = 1 << 2,    // Binary: 0100 (Decimal 4)
    Delete  = 1 << 3,    // Binary: 1000 (Decimal 8)
    
    // You can also define helper combinations
    All     = Read | Write | Execute | Delete
}
```

Using Enum Classes as Bitfields introduces a binary operators to Enum Claasess and allows to use Enum class with as a regular C bitmask flag pattern
with & to check if Flag is set, |, ^, ~, 

For example:

```c++
A a1=A::x | A::y;
A a2=a1&A::y; // Turn on A::y only
ASSERT_EQ(A::y, a2);
a2^=A::x; // toggle A::x
ASSERT_EQ(A::x | A::y, a2);
A a3=a1&~A::x; // Turn off A::X
ASSERT_EQ(A::y, a3);

B b1=B::x | B::y;
B b2=b1&B::y;
ASSERT_EQ(B::y, b2);
b2^=B::x;
ASSERT_EQ(B::x | B::y, b2);
B b3=b1&~B::x;
ASSERT_EQ(B::y, b3);
```

## Can i use std::bitset instead?

std::bitset doesn't provide the type safety guranties as the enum class.

```C++
std::bitset<8> data("10101100"); // Our main bitset
    
// Create a mask for the bits you want to check (e.g., positions 3 and 5)
std::bitset<8> mask;
mask.set(3);
mask.set(5);

// Check if ALL bits in the mask are set in 'data'
bool all_set = (data & mask) == mask;
std::cout << "Are bits 3 and 5 both set? " << std::boolalpha << all_set << "\n";

// Check if AT LEAST ONE bit from the mask is set in 'data'
bool any_set = (data & mask).any();
std::cout << "Is either bit 3 or 5 set? " << any_set << "\n";
```

The Flag names are lost std::bitset doesn't provide the type safety guranties as the enum class.

# Design

We want to allow the developers to opt in into bitmask behavior, there for we provide concept BitmaskLike for extending the enums with tenarry operators.
This design is based on the https://andreasfertig.com/blog/2024/01/cpp20-concepts-applied/ evolution.

Allow users to add Binary operators to their Enum Class.
